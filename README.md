### Getting Started
The application can be started using Maven.

> `mvn clean install`
>
> `mvn spring-boot:run`

### Facilities
- Port - 8080
- Swagger - http://localhost:8080/swagger-ui/index.html
- H2 - http://localhost:8080/h2-console
